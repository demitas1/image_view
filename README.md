# Image view

## Install

```
pip install -r requrements.txt
```

## Usage

- image view

```
python image_view.py <directory>
```

  - LMB / Right:  Next image
  - RMB / Left:   Previous image
  - MMB / Space:   Context Menu
  - R:  Toggle random order
  - H:  Toggle H-flip
  - Ctrl + C: Copy image
  - Q:  Quit
